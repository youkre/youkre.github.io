## 向量点积

假设我们只有一个样本，特征向量有三个值：<span class="math"><math><mo stretchy="false">(</mo><msub><mi>x</mi><mn>1</mn></msub><mo>,</mo><msub><mi>x</mi><mn>2</mn></msub><mo>,</mo><msub><mi>x</mi><mn>3</mn></msub><mo stretchy="false">)</mo></math></span>，每个输入特征 <span class="math"><math><msub><mi>x</mi><mi>i</mi></msub></math></span> 有一个权重值 <span class="math"><math><msub><mi>w</mi><mi>i</mi></msub></math></span> 相关联，表示每个 <span class="math"><math><msub><mi>x</mi><mi>i</mi></msub></math></span> 在最终结果的重要性，可以这样计算：

## 损失函数

逻辑回归的输出结果是一个概率，那么我们希望这个概率与真实结果越接近越好。这里需要用到伯努利分布。

伯努利试验只有 1 次，结果要么成功（ <span class="math"><math><mi>k</mi><mo>=</mo><mn>1</mn></math></span> ）、要么失败（ <span class="math"><math><mi>k</mi><mo>=</mo><mn>0</mn></math></span> ）

- 成功 1 次：概率是 <span class="math"><math><mi>p</mi></math></span>（乘以 1 次 <span class="math"><math><mi>p</mi></math></span>）→ 写出来就是 <span class="math"><math><msup><mi>p</mi><mn>1</mn></msup></math></span>
- 成功 0 次：概率是 <span class="math"><math><mn>1</mn><mo>−</mo><mi>p</mi></math></span>（乘以 1 次 <span class="math"><math><mn>1</mn><mo>−</mo><mi>p</mi></math></span>）→ 写出来就是 <span class="math"><math><mo stretchy="false">(</mo><mn>1</mn><mo>−</mo><mi>p</mi><msup><mo stretchy="false">)</mo><mn>1</mn></msup></math></span>

它的概率质量函数分段写成：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>Y</mi><mo>=</mo><mi>y</mi><mo stretchy="false">)</mo><mo>=</mo><mrow><mo>{</mo><mtable><mtr><mtd style="text-align: left;justify-items: start;padding-right: 0;"><mi>p</mi><mo rspace="0">,</mo></mtd><mtd style="text-align: left;justify-items: start;padding-right: 0;padding-left:1em;"><mtext>如果 </mtext><mi>y</mi><mo>=</mo><mn>1</mn></mtd></mtr><mtr><mtd style="text-align: left;justify-items: start;padding-right: 0;"><mn>1</mn><mo>−</mo><mi>p</mi><mo rspace="0">,</mo></mtd><mtd style="text-align: left;justify-items: start;padding-right: 0;padding-left:1em;"><mtext>如果 </mtext><mi>y</mi><mo>=</mo><mn>0</mn></mtd></mtr></mtable><mo>⁣</mo></mrow></math>
</div>

或者写成更紧凑的“指示函数”形式（用 <span class="math"><math><mi>y</mi></math></span> 的取值作为指数）：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>Y</mi><mo>=</mo><mi>y</mi><mo stretchy="false">)</mo><mo>=</mo><msup><mi>p</mi><mi>y</mi></msup><mo stretchy="false">(</mo><mn>1</mn><mo>−</mo><mi>p</mi><msup><mo stretchy="false">)</mo><mrow><mn>1</mn><mo lspace="0" rspace="0">−</mo><mi>y</mi></mrow></msup><mo>,</mo><mspace width="1em"/><mtext>其中 </mtext><mi>y</mi><mo>∈</mo><mo stretchy="false">{</mo><mn>0</mn><mo>,</mo><mn>1</mn><mo stretchy="false">}</mo></math>
</div>


扩展到更一般的情况：二项分布。

如果是做 <span class="math"><math><mi>n</mi></math></span> 次试验，成功 <span class="math"><math><mi>k</mi></math></span> 次：

- 概率连乘：成功 <span class="math"><math><mi>k</mi></math></span> 次 → 乘 <span class="math"><math><mi>k</mi></math></span> 次 <span class="math"><math><mi>p</mi></math></span> → <span class="math"><math><msup><mi>p</mi><mi>k</mi></msup></math></span>
- 失败连乘：失败 <span class="math"><math><mi>n</mi><mo>−</mo><mi>k</mi></math></span> 次 → 乘 <span class="math"><math><mi>n</mi><mo>−</mo><mi>k</mi></math></span> 次 <span class="math"><math><mn>1</mn><mo>−</mo><mi>p</mi></math></span> → <span class="math"><math><mo stretchy="false">(</mo><mn>1</mn><mo>−</mo><mi>p</mi><msup><mo stretchy="false">)</mo><mrow><mi>n</mi><mo lspace="0" rspace="0">−</mo><mi>k</mi></mrow></msup></math></span>
- 排列方式：这 <span class="math"><math><mi>k</mi></math></span> 次成功可以出现在 <span class="math"><math><mi>n</mi></math></span> 个位置中的任意 <span class="math"><math><mi>k</mi></math></span> 个 → 乘以 <span class="math"><math><mrow><mo>(</mo><mfrac linethickness="0"><mi>n</mi><mi>k</mi></mfrac><mo>)</mo></mrow></math></span>

当试验次数 <span class="math"><math><mi>n</mi><mo>=</mo><mn>1</mn></math></span> 时，组合数退化为 1，剩下的就是纯粹的“概率连乘”本身。这就是伯努利分布。

在机器学习里，我们要预测的 <span class="math"><math><mi>y</mi></math></span> 是真实标签，取值 0 或 1，但“概率参数 <span class="math"><math><mi>p</mi></math></span> 在这里是什么？

## 多项逻辑回归的损失函数

多分类不能用伯努利分布，因为伯努利只管“是/否”两种结果。多分类需要用“分类分布”（Categorical Distribution），这是伯努利在多个类别上的推广。

| | 伯努利分布 | 分类分布 |

| **参数** | 一个参数 <span class="math"><math><mi>p</mi></math></span>，表示 <span class="math"><math><mi>y</mi><mo>=</mo><mn>1</mn></math></span> 的概率 | K 个参数 <span class="math"><math><msub><mi>p</mi><mn>1</mn></msub><mo>,</mo><msub><mi>p</mi><mn>2</mn></msub><mo>,</mo><mi>.</mi><mi>.</mi><mi>.</mi><mo>,</mo><msub><mi>p</mi><mi>K</mi></msub></math></span>，表示每个类别的概率，且 <span class="math"><math><munderover><mo lspace="0">∑</mo><mrow><mi>i</mi><mo lspace="0" rspace="0">=</mo><mn>1</mn></mrow><mi>K</mi></munderover><msub><mi>p</mi><mi>i</mi></msub><mo>=</mo><mn>1</mn></math></span> |
