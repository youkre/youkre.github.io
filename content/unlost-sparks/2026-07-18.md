## 习题一

某聚会上,被邀请的人逐一到达。在等待其他人到达时，已到达的人们通过互相对比自己的生日来自娱自乐。设 X 表示生日匹配成功所需的人数，即在第X个人到达之前，没有两个人的生日相同，但当第 X 个人到达时正好有一对生日相匹配。求 X 的概率质量函数。

### 把文字翻译成事件

题目定义的随机变量 X 是：“生日匹配成功所需的人数”。

这个定义包含了两个必须同时成立的条件，用集合论的语言写出来就是：

事件 {X = n} = {前 n-1 个人生日全不相同} ∩ {第 n 个人的生日恰好等于前 n-1 个人中某一个人的生日}

这里的关键是：第 n 个人到达时，只产生“一对”匹配。这意味着第 n 个人的生日需要从前 n-1 个人的生日中选一个。

因为随机变量 X 的取值 n，取决于它前面 n-1 个人的状态。也就是说，我们要先保证前面 n-1 个人“无事发生”，然后再看第 n 个人“引发事件”。

所以概率质量函数的推导路径是：

P(X = n) = P(前 n-1 人全不同 且 第 n 人与前 n-1 人中某一人相同)

### 用乘法原则直接计算

设一年有 365 天（忽略闰年）。

前 n-1 个人生日全不同的排列数：
第1个人有 365 种选择，第 2 个人有 36 4种...第 n-1 个人有 365 - (n-1) + 1 = 365 - n + 2 种选择。
所以排列数为：365 × 364 × ... × (365 - n + 2)。

第 n 个人与前面某一人匹配的方式：
 第 n 个人的生日必须等于前 n-1 个人中任意一个的生日，所以只有 (n-1) 种选择。

总样本空间（所有可能的生日分配方式）：
n 个人每个人有365种选择，总数是 365ⁿ。

因此，概率质量函数为：

P(X = n) = [365 × 364 × ... × (365 - n + 2) × (n - 1)] / 365ⁿ

其中 n 的取值范围是：2 ≤ n ≤ 366（因为最多366人必定匹配，超过366人时，前365人不可能全不同，所以概率为0）。

### 验证（取极端值）

- 当 n=2 时，P(X=2) = (365 × 1) / 365² = 1/365。这符合直觉：两个人恰好生日相同的概率就是 1/365。
- 当 n=366 时，P(X=366) = (365 × 364 × ... × 1 × 365) / 365³⁶⁶。这个值极小，表示前365人全不同（占满所有天数）且最后一人必须匹配某一天。

## 习题二

(a) 独立地进行伯努利试验（试验成功的概率为 1/2），直到至少有一次试验成功。求试验实施次数的概率质量函数。

### 用集合论语言描述事件

设每次试验的结果为 <span class="math"><math><msub><mi>ω</mi><mi>i</mi></msub><mo>∈</mo><mrow><mtext>成功</mtext><mo stretchy="false">(</mo><mi>S</mi><mo stretchy="false">)</mo><mi>，</mi><mtext>失败</mtext><mo stretchy="false">(</mo><mi>F</mi><mo stretchy="false">)</mo></mrow></math></span>，成功的概率 <span class="math"><math><mi>p</mi><mo>=</mo><mfrac><mn>1</mn><mn>2</mn></mfrac></math></span>。

随机变量 X 定义为：“首次成功所需的试验次数”。

事件 {X = n} 对应样本空间中的一个特定集合：

<div class="math">
<math display="block"><mo stretchy="false">{</mo><mi>X</mi><mo>=</mo><mi>n</mi><mo stretchy="false">}</mo><mo>=</mo><mo stretchy="false">{</mo><msub><mi>ω</mi><mn>1</mn></msub><mo>=</mo><mi>F</mi><mo>,</mo><msub><mi>ω</mi><mn>2</mn></msub><mo>=</mo><mi>F</mi><mo>,</mo><mi>.</mi><mi>.</mi><mi>.</mi><mo>,</mo><msub><mi>ω</mi><mrow><mi>n</mi><mo lspace="0" rspace="0">−</mo><mn>1</mn></mrow></msub><mo>=</mo><mi>F</mi><mo>,</mo><msub><mi>ω</mi><mi>n</mi></msub><mo>=</mo><mi>S</mi><mo stretchy="false">}</mo></math>
</div>

这是一个单点事件——样本空间中只有一个具体的结果序列满足这个条件（因为每次试验的结果是确定的：前 n-1 次必须全失败，第 n 次必须成功）。

### 计算概率

由于各次试验独立，概率为各次结果概率的乘积：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>n</mi><mo stretchy="false">)</mo><mo>=</mo><mi>P</mi><mo stretchy="false">(</mo><mi>F</mi><mo stretchy="false">)</mo><mo lspace="0.2222em" rspace="0.2222em">×</mo><mi>P</mi><mo stretchy="false">(</mo><mi>F</mi><mo stretchy="false">)</mo><mo lspace="0.2222em" rspace="0.2222em">×</mo><mi>.</mi><mi>.</mi><mi>.</mi><mi>×</mi><mi>P</mi><mo stretchy="false">(</mo><mi>F</mi><mo stretchy="false">)</mo><mo lspace="0.2222em" rspace="0.2222em">×</mo><mi>P</mi><mo stretchy="false">(</mo><mi>S</mi><mo stretchy="false">)</mo></math>
</div>

其中前 n-1 项为 P(F) = 1/2，最后一项为 P(S) = 1/2。

因此：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>n</mi><mo stretchy="false">)</mo><mo>=</mo><msup><mfrac><mn>1</mn><mn>2</mn></mfrac><mrow><mi>n</mi><mo lspace="0" rspace="0">−</mo><mn>1</mn></mrow></msup><mo lspace="0.2222em" rspace="0.2222em">×</mo><mfrac><mn>1</mn><mn>2</mn></mfrac><mo>=</mo><msup><mfrac><mn>1</mn><mn>2</mn></mfrac><mi>n</mi></msup></math>
</div>

### 验证

- n = 1：<span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mn>1</mn><mo stretchy="false">)</mo><mo>=</mo><mo stretchy="false">(</mo><mfrac><mn>1</mn><mn>2</mn></mfrac><msup><mo stretchy="false">)</mo><mn>1</mn></msup><mo>=</mo><mfrac><mn>1</mn><mn>2</mn></mfrac></math></span>。第一次就成功的概率为 1/2，正确。
- n = 2：<span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mn>2</mn><mo stretchy="false">)</mo><mo>=</mo><mo stretchy="false">(</mo><mfrac><mn>1</mn><mn>2</mn></mfrac><msup><mo stretchy="false">)</mo><mn>2</mn></msup><mo>=</mo><mfrac><mn>1</mn><mn>4</mn></mfrac></math></span>。第一次失败、第二次成功的概率为 <span class="math"><math><mfrac><mn>1</mn><mn>2</mn></mfrac><mo lspace="0.2222em" rspace="0.2222em">×</mo><mfrac><mn>1</mn><mn>2</mn></mfrac><mo>=</mo><mfrac><mn>1</mn><mn>4</mn></mfrac></math></span>，正确。
- 所有概率求和：<span class="math"><math><munderover><mo lspace="0" rspace="0">∑</mo><mrow><mi>n</mi><mo lspace="0" rspace="0">=</mo><mn>1</mn></mrow><mi>∞</mi></munderover><mo stretchy="false">(</mo><mfrac><mn>1</mn><mn>2</mn></mfrac><msup><mo stretchy="false">)</mo><mi>n</mi></msup><mo>=</mo><mn>1</mn></math></span>，归一化成立。

### 错误思路

<div class="math">
<math display="block"><mrow><mo>(</mo><mfrac linethickness="0"><mi>n</mi><mrow><mi>n</mi><mo>−</mo><mn>1</mn></mrow></mfrac><mo>)</mo></mrow><mo stretchy="false">(</mo><mfrac><mn>1</mn><mn>2</mn></mfrac><msup><mo stretchy="false">)</mo><mrow><mi>n</mi><mo lspace="0" rspace="0">−</mo><mn>1</mn></mrow></msup><mo lspace="0.2222em" rspace="0.2222em">×</mo><mfrac><mn>1</mn><mn>2</mn></mfrac></math>
</div>

解析中出现了 <span class="math"><math><mrow><mo>(</mo><mfrac linethickness="0"><mi>n</mi><mrow><mi>n</mi><mo>−</mo><mn>1</mn></mrow></mfrac><mo>)</mo></mrow></math></span>，这通常是二项分布中的计数因子——表示在 n 次试验中选哪 n-1 次作为失败的位置。

但二项分布描述的模型是：固定试验次数 n，问其中恰好有 k 次成功。在这种情况下，成功可以出现在任意位置，所以需要组合数来计数不同排列。

而本题的模型是：试验次数不固定，直到首次成功为止。在这种情况下，成功的位置被固定在了第 n 次，前面 n-1 次的位置全部被固定为失败，因此没有选择的空间，组合数 \binom{n}{n-1} = n 不适用。

换句话说：
- 二项分布：在 n 次试验中，成功可以落在 n 个位置中的任意 k 个 → 需要组合数。
- 几何分布（本题）：首次成功必须落在第 n 次，前面的位置全部固定 → 不需要组合数。

总结对比

| 模型 | 成功位置 | 失败位置 | 是否需要组合数 |
|---|---|---|---|
| 二项分布：固定 n 次，恰好 k 次成功 | 任意 k 个位置 | 任意 n-k 个位置 | 需要 <span class="math"><math><mrow><mo>(</mo><mfrac linethickness="0"><mi>n</mi><mi>k</mi></mfrac><mo>)</mo></mrow></math></span> |
| 几何分布：直到首次成功为止 | 固定在第 n 次 | 固定在前 n-1 次 | 不需要组合数 |

随机变量 X = "直到至少有一次成功且至少有一次失败所需的试验次数"。

事件 {X = n} 的含义是：

- 前 n-1 次试验不能同时包含 S 和 F（否则早就结束了）；
- 第 n 次试验产生另一种结果，使得前 n 次中 S 和 F 都至少出现一次。

因此，前 n-1 次必须全部相同，只有两种可能：
- 前 n-1 次全为 S，第 n 次为 F；
- 前 n-1 次全为 F，第 n 次为 S。

这两个事件互斥（不可能同时发生），因此概率相加。

## 习题三

设 X 是一个随机变量，且 X 的累积分布函数为 F，再设 <span class="math"><math><mi>Y</mi><mo>=</mo><mi>μ</mi><mo>+</mo><mi>σ</mi><mi>X</mi></math></span>，其中 <span class="math"><math><mi>μ</mi></math></span> 和 <span class="math"><math><mi>σ</mi></math></span> 为实数，且 <span class="math"><math><mi>σ</mi><mo>&gt;</mo><mn>0</mn></math></span>。利用 F 求 Y 的概率质量函数。

### 定位问题所在

正确的写法是：设 <span class="math"><math><mi>y</mi></math></span> 为任意实数，则

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>Y</mi><mo>=</mo><mi>y</mi><mo stretchy="false">)</mo><mo>=</mo><mi>P</mi><mo stretchy="false">(</mo><mi>μ</mi><mo>+</mo><mi>σ</mi><mi>X</mi><mo>=</mo><mi>y</mi><mo stretchy="false">)</mo><mo>=</mo><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mfrac><mrow><mi>y</mi><mo>−</mo><mi>μ</mi></mrow><mi>σ</mi></mfrac><mo stretchy="false">)</mo></math>
</div>

也就是说，要求 <span class="math"><math><mi>Y</mi></math></span> 在某个具体取值 <span class="math"><math><mi>y</mi></math></span> 处的概率，等价于求 <span class="math"><math><mi>X</mi></math></span> 在对应取值 <span class="math"><math><mfrac><mrow><mi>y</mi><mo>−</mo><mi>μ</mi></mrow><mi>σ</mi></mfrac></math></span> 处的概率。

### 用分布函数 <span class="math"><math><mi>F</mi></math></span> 表示

题目给出的条件是：<span class="math"><math><mi>X</mi></math></span> 的累积分布函数为 <span class="math"><math><mi>F</mi></math></span>。

累积分布函数的定义是：

<div class="math">
<math display="block"><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>≤</mo><mi>x</mi><mo stretchy="false">)</mo></math>
</div>

但概率质量函数（PMF）要求的是 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>x</mi><mo stretchy="false">)</mo></math></span>，而不是 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>≤</mo><mi>x</mi><mo stretchy="false">)</mo></math></span>。

这里就出现了这道题的关键点：如果只知道累积分布函数 <span class="math"><math><mi>F</mi></math></span>，能否唯一确定 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>x</mi><mo stretchy="false">)</mo></math></span>？

答案取决于 <span class="math"><math><mi>X</mi></math></span> 的类型，这里我们假设 <span class="math"><math><mi>X</mi></math></span> 是离散型随机变量，概率质量函数可以通过累积分布函数的跳跃来恢复：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>−</mo><mi>F</mi><mo stretchy="false">(</mo><msup><mi>x</mi><mo>−</mo></msup><mo stretchy="false">)</mo></math>
</div>

其中 <span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><msup><mi>x</mi><mo>−</mo></msup><mo stretchy="false">)</mo><mo>=</mo><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>t</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><msup><mi>x</mi><mo>−</mo></msup></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>t</mi><mo stretchy="false">)</mo></math></span>，即 <span class="math"><math><mi>F</mi></math></span> 在 <span class="math"><math><mi>x</mi></math></span> 处的左极限。

因此：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>Y</mi><mo>=</mo><mi>y</mi><mo stretchy="false">)</mo><mo>=</mo><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mfrac><mrow><mi>y</mi><mo>−</mo><mi>μ</mi></mrow><mi>σ</mi></mfrac><mo stretchy="false">)</mo><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><mfrac><mrow><mi>y</mi><mo>−</mo><mi>μ</mi></mrow><mi>σ</mi></mfrac><mo stretchy="false">)</mo><mo>−</mo><mi>F</mi><mo stretchy="false">(</mo><mo stretchy="false">(</mo><mfrac><mrow><mi>y</mi><mo>−</mo><mi>μ</mi></mrow><mi>σ</mi></mfrac><msup><mo stretchy="false">)</mo><mo>−</mo></msup><mo stretchy="false">)</mo></math>
</div>

## 习题四

设 <span class="math"><math><mi>n</mi></math></span> 是正整数， 当 <span class="math"><math><mn>0</mn><mo>≤</mo><mi>x</mi><mo>≤</mo><mi>n</mi></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mfrac displaystyle="true"><mrow><mo stretchy="false">⌊</mo><mi>x</mi><mo stretchy="false">⌋</mo></mrow><mi>n</mi></mfrac></math></span>；当 <span class="math"><math><mi>x</mi><mo>&lt;</mo><mn>0</mn></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>0</mn></math></span>；当  <span class="math"><math><mi>x</mi><mo>&gt;</mo><mi>n</mi></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span>。其中 <span class="math"><math><mo stretchy="false">⌊</mo><mi>x</mi><mo stretchy="false">⌋</mo></math></span> 表示不大于 <span class="math"><math><mi>x</mi></math></span> 的最大整数。证明：<span class="math"><math><mi>F</mi></math></span> 是一个累积分布函数并计算它所对应的概率质量函数。

### 证明 F 是一个累积分布函数

累积分布函数需要满足三个条件：

1. 非降
2. 右连续
3. 极限性质：<span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>x</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><mo lspace="0" rspace="0">−</mo><mi>∞</mi></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>0</mn></math></span>，<span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>x</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><mo lspace="0" rspace="0">+</mo><mi>∞</mi></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span>

### 非降

设 <span class="math"><math><msub><mi>x</mi><mn>1</mn></msub><mo>&lt;</mo><msub><mi>x</mi><mn>2</mn></msub></math></span>。

- 若两者在同一区间 <span class="math"><math><mo stretchy="false">[</mo><mi>k</mi><mo>,</mo><mi>k</mi><mo>+</mo><mn>1</mn><mo stretchy="false">)</mo></math></span> 内，则 <span class="math"><math><mo stretchy="false">⌊</mo><msub><mi>x</mi><mn>1</mn></msub><mo stretchy="false">⌋</mo><mo>=</mo><mo stretchy="false">⌊</mo><msub><mi>x</mi><mn>2</mn></msub><mo stretchy="false">⌋</mo></math></span>，所以 <span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><msub><mi>x</mi><mn>1</mn></msub><mo stretchy="false">)</mo><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><msub><mi>x</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>。
- 若 <span class="math"><math><msub><mi>x</mi><mn>1</mn></msub></math></span> 在区间 <span class="math"><math><mo stretchy="false">[</mo><mi>k</mi><mo>,</mo><mi>k</mi><mo>+</mo><mn>1</mn><mo stretchy="false">)</mo></math></span>，<span class="math"><math><msub><mi>x</mi><mn>2</mn></msub></math></span> 在区间 <span class="math"><math><mo stretchy="false">[</mo><mi>k</mi><mo>+</mo><mn>1</mn><mo>,</mo><mi>k</mi><mo>+</mo><mn>2</mn><mo stretchy="false">)</mo></math></span>，则 <span class="math"><math><mo stretchy="false">⌊</mo><msub><mi>x</mi><mn>1</mn></msub><mo stretchy="false">⌋</mo><mo>=</mo><mi>k</mi></math></span>，<span class="math"><math><mo stretchy="false">⌊</mo><msub><mi>x</mi><mn>2</mn></msub><mo stretchy="false">⌋</mo><mo>=</mo><mi>k</mi><mo>+</mo><mn>1</mn></math></span>，所以 <span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><msub><mi>x</mi><mn>1</mn></msub><mo stretchy="false">)</mo><mo>=</mo><mi>k</mi><mo lspace="0" rspace="0">/</mo><mi>n</mi><mo>&lt;</mo><mo stretchy="false">(</mo><mi>k</mi><mo>+</mo><mn>1</mn><mo stretchy="false">)</mo><mo lspace="0" rspace="0">/</mo><mi>n</mi><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><msub><mi>x</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>。
- 若跨越整数点，取整数值单调不降。

综上，当 <span class="math"><math><msub><mi>x</mi><mn>1</mn></msub><mo>&lt;</mo><msub><mi>x</mi><mn>2</mn></msub></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><msub><mi>x</mi><mn>1</mn></msub><mo stretchy="false">)</mo><mo>≤</mo><mi>F</mi><mo stretchy="false">(</mo><msub><mi>x</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>，满足非降性。

### 右连续

对于任意实数 <span class="math"><math><mi>x</mi></math></span>，需要验证 <span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>t</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><msup><mi>x</mi><mo>+</mo></msup></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>t</mi><mo stretchy="false">)</mo><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span>。

- 若 <span class="math"><math><mi>x</mi></math></span> 不是整数，则在 <span class="math"><math><mi>x</mi></math></span> 的一个足够小的右邻域内，<span class="math"><math><mo stretchy="false">⌊</mo><mi>t</mi><mo stretchy="false">⌋</mo><mo>=</mo><mo stretchy="false">⌊</mo><mi>x</mi><mo stretchy="false">⌋</mo></math></span>，因此 <span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>t</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><msup><mi>x</mi><mo>+</mo></msup></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>t</mi><mo stretchy="false">)</mo><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span>。
- 若 <span class="math"><math><mi>x</mi></math></span> 是整数，设 <span class="math"><math><mi>x</mi><mo>=</mo><mi>k</mi></math></span>（<span class="math"><math><mn>0</mn><mo>≤</mo><mi>k</mi><mo>≤</mo><mi>n</mi></math></span>）。当 <span class="math"><math><mi>t</mi><mo stretchy="false">→</mo><msup><mi>k</mi><mo>+</mo></msup></math></span> 时，<span class="math"><math><mi>t</mi><mo>∈</mo><mo stretchy="false">(</mo><mi>k</mi><mo>,</mo><mi>k</mi><mo>+</mo><mn>1</mn><mo stretchy="false">)</mo></math></span>，所以 <span class="math"><math><mo stretchy="false">⌊</mo><mi>t</mi><mo stretchy="false">⌋</mo><mo>=</mo><mi>k</mi></math></span>，因此 <span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>t</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><msup><mi>k</mi><mo>+</mo></msup></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>t</mi><mo stretchy="false">)</mo><mo>=</mo><mi>k</mi><mo lspace="0" rspace="0">/</mo><mi>n</mi><mo>=</mo><mi>F</mi><mo stretchy="false">(</mo><mi>k</mi><mo stretchy="false">)</mo></math></span>。注意 <span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>k</mi><mo stretchy="false">)</mo><mo>=</mo><mo stretchy="false">⌊</mo><mi>k</mi><mo stretchy="false">⌋</mo><mo lspace="0" rspace="0">/</mo><mi>n</mi><mo>=</mo><mi>k</mi><mo lspace="0" rspace="0">/</mo><mi>n</mi></math></span>，所以右连续性成立。

在边界处同样成立：<span class="math"><math><mi>x</mi><mo>=</mo><mn>0</mn></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mn>0</mn><mo stretchy="false">)</mo><mo>=</mo><mn>0</mn></math></span>，右极限也为0；<span class="math"><math><mi>x</mi><mo>=</mo><mi>n</mi></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>n</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span>，右极限也为1（因为 <span class="math"><math><mi>x</mi><mo>&gt;</mo><mi>n</mi></math></span> 时函数值为1）。

### 极限性质

- 当 <span class="math"><math><mi>x</mi><mo>&lt;</mo><mn>0</mn></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>0</mn></math></span>，因此 <span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>x</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><mo lspace="0" rspace="0">−</mo><mi>∞</mi></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>0</mn></math></span>。
- 当 <span class="math"><math><mi>x</mi><mo>&gt;</mo><mi>n</mi></math></span> 时，<span class="math"><math><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span>，因此 <span class="math"><math><munder><mo movablelimits="true" lspace="0" rspace="0.1667em">lim</mo><mrow><mi>x</mi><mo stretchy="false" lspace="0" rspace="0">→</mo><mo lspace="0" rspace="0">+</mo><mi>∞</mi></mrow></munder><mi>F</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span>。

### 计算对应的概率质量函数

### 集合论视角：这个 PMF 的样本空间是什么？

这个 PMF 的支撑集只有 9 个点（\( j = 1, 2, \ldots, 9 \)），而不是像几何分布那样有无穷多个点。

对应的样本空间 \( \Omega \) 可以理解为：从某个实际数据集中随机抽取一个数，观察其首位数字。

- 样本空间 \( \Omega = \{1, 2, \ldots, 9\} \)
- 每个结果 \( \omega = j \) 表示"首位数字为 j"
- 事件 \( \{D = j\} \) 就是单元素子集 \( \{j\} \)
- 概率赋值 \( P(\{j\}) = \lg((j+1)/j) \)

这个概率赋值不是均匀的（不像上一题的 \( 1/n \)），但它仍然满足概率公理，因为所有单点概率之和为 1。

\( P(X = 2) = p_1 \) 意味着"从第 1 关进入第 2 关"这一事件本身就被计为 \( X = 2 \)。

以此类推，\( P(X = 7) = p_1 \times p_2 \times \dots \times p_6 \) 意味着"到达第 7 关"（即成功通过前 6 关）就是 \( X = 7 \)。

### 补上 \( X = 1 \) 的情况

根据题意，Bob 从第 1 关开始，所以他已经"达到"了第 1 关。如果他在第 1 关失败（即未能以概率 \( p_1 \) 进入第 2 关），那么他达到的最高关数就是 第 1 关。

因此：

\[
P(X = 1) = 1 - p_1
\]

### 重新整理完整的 PMF

对于 \( 2 \leq j \leq 6 \)：

要使得最高关数为 \( j \)，Bob 必须：
- 成功通过第 1 关到第 \( j-1 \) 关（即到达第 j 关），
- 且未能通过第 j 关（即未能进入第 j+1 关）。

因此：

\[
P(X = j) = (p_1 \times p_2 \times \dots \times p_{j-1}) \times (1 - p_j)
\]

其中定义空乘积 \( p_1 \times \dots \times p_{0} = 1 \)（当 \( j=1 \) 时的情况单独处理）。

对于 \( j = 7 \)：

要使得最高关数为 7，Bob 必须成功通过第 1 关到第 6 关（即到达第 7 关）。此时游戏结束（没有第 8 关），不需要再乘以 \( (1 - p_7) \)。

因此：

\[
P(X = 7) = p_1 \times p_2 \times \dots \times p_6
\]

### 汇总为分段表达式

\[
P(X = j) = 
\begin{cases}
1 - p_1, & j = 1 \\[6pt]
\left(\prod_{i=1}^{j-1} p_i\right)(1 - p_j), & 2 \leq j \leq 6 \\[6pt]
\prod_{i=1}^{6} p_i, & j = 7
\end{cases}
\]

### 验证归一性

将所有概率相加：

\[
\sum_{j=1}^{7} P(X = j) = (1 - p_1) + \sum_{j=2}^{6} \left[\left(\prod_{i=1}^{j-1} p_i\right)(1 - p_j)\right] + \prod_{i=1}^{6} p_i
\]

这是一个望远镜求和（telescoping sum）：

- 第一项：\( 1 - p_1 \)
- 第二项（\( j=2 \)）：\( p_1(1 - p_2) \)
- 第三项（\( j=3 \)）：\( p_1 p_2 (1 - p_3) \)
- ...
- 最后一项：\( p_1 p_2 \dots p_6 \)

展开后，中间项全部抵消，只剩下 1。归一性成立。

### 错误的解析

\[
\begin{aligned}
P(X = 2) &= p_1 \\
P(X = 3) &= p_1 \times p_2 \\
P(X = 4) &= p_1 \times p_2 \times p_3 \\
P(X = 5) &= p_1 \times p_2 \times p_3 \times p_4 \\
P(X = 6) &= p_1 \times p_2 \times p_3 \times p_4 \times p_5 \\
P(X = 7) &= p_1 \times p_2 \times p_3 \times p_4 \times p_5 \times p_6
\end{aligned}
\]

问题在于：

对于 \( 2 \leq j \leq 6 \)，必须乘以 \( (1 - p_j) \) 来表示"在第 j 关失败"。

漏掉了 \( X = 1 \) 和每个中间关卡处的"失败因子"\( (1 - p_j) \)。这个模式在之前的几何分布题目中也出现过："首次成功"需要乘以失败概率来表示"在此之前一直失败"——本题同理，只是换成了"首次失败"。

### 用全概率公式建立证明

设 \( C \) 为抛硬币的结果：

\[
C = 
\begin{cases}
1, & \text{正面朝上（概率为 } p \text{）} \\
0, & \text{反面朝上（概率为 } 1-p \text{）}
\end{cases}
\]

给定 \( C = 1 \) 时，\( X \) 的分布由 \( F_1 \) 描述；给定 \( C = 0 \) 时，\( X \) 的分布由 \( F_2 \) 描述。

即：

<div class="math">
<math display="block"><mtable displaystyle="true" scriptlevel="0"><mtr><mtd style="text-align: right;justify-items: end;padding-right: 0;"><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>≤</mo><mi>x</mi><mo>∣</mo><mi>C</mi><mo>=</mo><mn>1</mn><mo stretchy="false">)</mo><mo>=</mo><msub><mi>F</mi><mn>1</mn></msub><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></mtd></mtr><mtr><mtd style="text-align: right;justify-items: end;padding-right: 0;"><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>≤</mo><mi>x</mi><mo>∣</mo><mi>C</mi><mo>=</mo><mn>0</mn><mo stretchy="false">)</mo><mo>=</mo><msub><mi>F</mi><mn>2</mn></msub><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></mtd></mtr></mtable></math>
</div>

现在对事件 \( \{X \leq x\} \) 使用全概率公式（按 \( C \) 的取值划分样本空间）：

\[
P(X \leq x) = P(X \leq x \mid C = 1) \cdot P(C = 1) + P(X \leq x \mid C = 0) \cdot P(C = 0)
\]

代入已知条件：

\[
P(X \leq x) = F_1(x) \cdot p + F_2(x) \cdot (1-p)
\]

即：

\[
F_X(x) = pF_1(x) + (1-p)F_2(x) = F(x)
\]

这正是题目要证明的结论。

### 混合分布（mixture distribution）：

\[
F = pF_1 + (1-p)F_2
\]

它表示以概率 \( p \) 从分布 \( F_1 \) 中抽样，以概率 \( 1-p \) 从分布 \( F_2 \) 中抽样。混合后的总体分布就是两者的加权平均。

如果 \( F_1 \) 和 \( F_2 \) 是离散型的，对应的 PMF 为：

\[
p_X(x) = p \cdot p_1(x) + (1-p) \cdot p_2(x)
\]

如果 \( F_1 \) 和 \( F_2 \) 是连续型的，对应的 PDF 为：

\[
f_X(x) = p \cdot f_1(x) + (1-p) \cdot f_2(x)
\]

混合分布在统计学中非常常见（例如：两个正态分布的混合、零膨胀泊松分布等）。

**解析**

这两道题考察的是同一个核心概念：概率质量函数的归一性。只需要判断这些值加起来是否等于 1，不需要构造出整个分布。

### 问题(a)

\( p(n) = 1/n \)，支撑集 \( n = 1, 2, 3, \ldots \)

如果 \( p(n) = 1/n \) 是一个有效的概率质量函数，那么必须满足：

\[
\sum_{n=1}^{\infty} p(n) = \sum_{n=1}^{\infty} \frac{1}{n} = 1
\]

这是调和级数（harmonic series）：

\[
\sum_{n=1}^{\infty} \frac{1}{n}
\]

根据级数理论，调和级数发散到无穷大：

\[
\sum_{n=1}^{\infty} \frac{1}{n} = \infty
\]

因为总和为 \( \infty \)，不等于 1（也不等于任何有限的数），所以不存在这样的概率质量函数。

### 问题(b)

\( p(n) = 1/n^2 \)，支撑集 \( n = 1, 2, 3, \ldots \)

如果 \( p(n) = 1/n^2 \) 是一个有效的概率质量函数，那么必须满足：

\[
\sum_{n=1}^{\infty} p(n) = \sum_{n=1}^{\infty} \frac{1}{n^2} = 1
\]

这是 \( p = 2 \) 的p-级数（p-series）

\[
\sum_{n=1}^{\infty} \frac{1}{n^2}
\]

根据级数理论，当 \( p > 1 \) 时，p-级数收敛。具体地：

\[
\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6} \approx 1.6449
\]

因为总和为 \( \pi^2/6 \approx 1.6449 \)，不等于 1，所以不存在这样的概率质量函数。
