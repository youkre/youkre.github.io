有向线段 <span class="math"><math><mover accent="true"><mrow><mi>P</mi><mi>Q</mi></mrow><mo>→</mo></mover></math></span> 的起点是 <span class="math"><math><mi>P</mi></math></span>，终点是 <span class="math"><math><mi>Q</mi></math></span>。
<span class="math"><math><mo stretchy="false" lspace="0" rspace="0">‖</mo><mi>P</mi><mi>Q</mi><mo stretchy="false" lspace="0" rspace="0">‖</mo></math></span> 表示线段长度。

如果多个有向线段的长度和方向都相同，则它们相等。
如果一组有向线段都等于一个特定的有向线段 <span class="math"><math><mover accent="true"><mrow><mi>P</mi><mi>Q</mi></mrow><mo>→</mo></mover></math></span>，则这组线段就是平面上的一个向量，表示成：

<div class="math">
<math display="block"><mi>𝐯</mi><mo>=</mo><mover accent="true"><mrow><mi>P</mi><mi>Q</mi></mrow><mo>→</mo></mover></math>
</div>

注意，向量代表一组有向线段，其中每条线段的长度和方向都相同。

起点在坐标轴原点的有向线段最方便表示一组相等的有向线段。v 的这个表示就在标准位置。

如果 <span class="math"><math><mi>𝐯</mi></math></span> 是平面的一个向量，起点是原点，终点 <span class="math"><math><mo stretchy="false">(</mo><msub><mi>v</mi><mn>1</mn></msub><mo>,</mo><msub><mi>v</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>，则 <span class="math"><math><mi>𝐯</mi></math></span> 的分量形式为 <span class="math"><math><mi>𝐯</mi><mo>=</mo><mo stretchy="false">(</mo><msub><mi>v</mi><mn>1</mn></msub><mo>,</mo><msub><mi>v</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>。<span class="math"><math><msub><mi>v</mi><mn>1</mn></msub></math></span> 和 <span class="math"><math><msub><mi>v</mi><mn>2</mn></msub></math></span> 是 <span class="math"><math><mi>𝐯</mi></math></span> 的两个分量。如果起点和终点都在原点，那么 <span class="math"><math><mi>𝐯</mi></math></span> 称为零向量，表示为 <span class="math"><math><mi>𝟎</mi><mo>=</mo><mo stretchy="false">(</mo><mn>0</mn><mo>,</mo><mn>0</mn><mo stretchy="false">)</mo></math></span>。

也就是说，如果两个向量相等，则它们的分量也相等。

如果有向线段 <span class="math"><math><mover accent="true"><mrow><mi>P</mi><mi>Q</mi></mrow><mo>→</mo></mover></math></span> 的起点是 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><msub><mi>p</mi><mn>1</mn></msub><mo>,</mo><msub><mi>p</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>，终点是 <span class="math"><math><mi>Q</mi><mo stretchy="false">(</mo><msub><mi>q</mi><mn>1</mn></msub><mo>,</mo><msub><mi>q</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>，则 <span class="math"><math><mover accent="true"><mrow><mi>P</mi><mi>Q</mi></mrow><mo>→</mo></mover></math></span> 的分量形式为 <span class="math"><math><mi>𝐯</mi><mo>=</mo><mo stretchy="false">(</mo><msub><mi>q</mi><mn>1</mn></msub><mo>−</mo><msub><mi>p</mi><mn>1</mn></msub><mo>,</mo><msub><mi>q</mi><mn>2</mn></msub><mo>−</mo><msub><mi>p</mi><mn>2</mn></msub><mo stretchy="false">)</mo></math></span>。

<span class="math"><math><mi>𝐯</mi></math></span> 的长度是：

<div class="math">
<math display="block"><mo stretchy="false" lspace="0" rspace="0">‖</mo><mi>𝐯</mi><mo stretchy="false" lspace="0" rspace="0">‖</mo><mo>=</mo><msqrt><mrow><mo stretchy="false">(</mo><msub><mi>q</mi><mn>1</mn></msub><mo>−</mo><msub><mi>p</mi><mn>1</mn></msub><msup><mo stretchy="false">)</mo><mn>2</mn></msup><mo>+</mo><mo stretchy="false">(</mo><msub><mi>q</mi><mn>2</mn></msub><mo>−</mo><msub><mi>p</mi><mn>2</mn></msub><msup><mo stretchy="false">)</mo><mn>2</mn></msup></mrow></msqrt></math>
</div>

<span class="math"><math><mi>𝐯</mi></math></span> 的长度也称为范数。如果 <span class="math"><math><mo stretchy="false" lspace="0" rspace="0">‖</mo><mi>𝐯</mi><mo>=</mo><mn>1</mn><mo stretchy="false" lspace="0" rspace="0">‖</mo></math></span>，那么 <span class="math"><math><mi>𝐯</mi></math></span> 就是单位向量。

当且仅当 <span class="math"><math><mi>𝐯</mi></math></span> 是零向量 <span class="math"><math><mi>𝟎</mi></math></span> 时，<span class="math"><math><mo stretchy="false" lspace="0" rspace="0">‖</mo><mi>𝐯</mi><mo>=</mo><mn>0</mn><mo stretchy="false" lspace="0" rspace="0">‖</mo></math></span>。
