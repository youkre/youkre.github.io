## 离散型随机变量的定义

这是根据 Introduction to Probability 3.2 节的翻译：

如果存在一个由值 <span class="math"><math><msub><mi>a</mi><mn>1</mn></msub><mo>,</mo><msub><mi>a</mi><mn>2</mn></msub><mo>,</mo><mo lspace="0">.</mo><mo lspace="0" rspace="0.1667em">.</mo><mo lspace="0">.</mo><mo>,</mo><msub><mi>a</mi><mi>n</mi></msub></math></span> 构成的有限列表，或者由值 <span class="math"><math><msub><mi>a</mi><mn>1</mn></msub><mo>,</mo><msub><mi>a</mi><mn>2</mn></msub><mo>,</mo><mo lspace="0">.</mo><mo lspace="0" rspace="0.1667em">.</mo><mo lspace="0" rspace="0">.</mo></math></span> 构成无限列表，存在某个 <span class="math"><math><mi>j</mi></math></span>，使得 <span class="math"><math><mi>X</mi><mo>=</mo><msub><mi>a</mi><mi>j</mi></msub></math></span> 这个事件的概率为 1，则称 X 为**离散型随机变量**。

如果 X 是离散型随机变量，并且由值 x 构成成的集合是有限或者可数无穷的，使得 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>x</mi><mo stretchy="false">)</mo><mo>&gt;</mo><mn>0</mn></math></span>，则称该集合 为 X 的**支撑**。 

## 离散型随机变量定义的解释

原文定义是这样写的：

A random variable <span class="math"><math><mi>X</mi></math></span> is said to be discrete if there is a finite list of values <span class="math"><math><mo stretchy="false">{</mo><msub><mi>a</mi><mn>1</mn></msub><mo>,</mo><msub><mi>a</mi><mn>2</mn></msub><mo>,</mo><mo lspace="0">.</mo><mo lspace="0" rspace="0.1667em">.</mo><mo lspace="0">.</mo><mo>,</mo><msub><mi>a</mi><mi>n</mi></msub><mo stretchy="false">}</mo></math></span> or an infinite list of values <span class="math"><math><mo stretchy="false">{</mo><msub><mi>a</mi><mn>1</mn></msub><mo>,</mo><msub><mi>a</mi><mn>2</mn></msub><mo>,</mo><mo lspace="0">.</mo><mo lspace="0" rspace="0.1667em">.</mo><mo lspace="0" rspace="0">.</mo><mo stretchy="false">}</mo></math></span> such that <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><msub><mi>a</mi><mi>j</mi></msub><mtext> </mtext><mtext>for some</mtext><mtext> </mtext><mi>j</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span>. If <span class="math"><math><mi>X</mi></math></span> is a discrete r.v., then the finite or countably infinite set of values <span class="math"><math><mi>x</mi></math></span> such that <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><mi>x</mi><mo stretchy="false">)</mo><mo>&gt;</mo><mn>0</mn></math></span> is called the support of <span class="math"><math><mi>X</mi></math></span>.

这里 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>X</mi><mo>=</mo><msub><mi>a</mi><mi>j</mi></msub><mtext> </mtext><mtext>for some</mtext><mtext> </mtext><mi>j</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn></math></span> 初看让人比较困惑，它表示的是：随机变量 <span class="math"><math><mi>X</mi></math></span> 的取值一定落在这个列表 <span class="math"><math><mo stretchy="false">{</mo><msub><mi>a</mi><mi>a</mi></msub><mo>,</mo><msub><mi>a</mi><mn>2</mn></msub><mo>,</mo><mo lspace="0">.</mo><mo lspace="0" rspace="0.1667em">.</mo><mo lspace="0" rspace="0">.</mo><mo stretchy="false">}</mo></math></span> 中的某一个值上，这个事件的概率是 1（即必然发生）

用集合语言写出来更清楚：

\[
P\left( X \in \{a_1, a_2, \ldots\} \right) = 1
\]

也就是说，\( X \) 以概率 1 取这个列表中的某个值。至于具体取哪一个，由概率质量函数 \( P(X = a_j) \) 来分配，但所有可能取值的概率之和必须等于 1。

原文这句话其实可以用更直观的方式重写：

> 存在一个有限或可数无穷的实数列表 \(\{a_1, a_2, \ldots\}\)，使得
> \[
> \sum_{j} P(X = a_j) = 1
> \]
> 也就是说，\( X \) 以概率 1 取这个列表中的某个值。

或者等价地：

> \( X \) 是离散型随机变量，如果存在一个有限或可数无穷的集合 \( A \subset \mathbb{R} \)，使得
> \[
> P(X \in A) = 1
> \]

## 支撑（support）与这个定义的关系

定义的第二句话是：

> 如果 \( X \) 是离散型随机变量，那么所有使得 \( P(X = x) > 0 \) 的 \( x \) 所组成的有限或可数无穷集合，称为 \( X \) 的支撑。

这两个定义的关系是：

1. **第一句话**定义了离散型随机变量：存在一个列表，\( X \) 以概率 1 取这个列表中的值。
2. **第二句话**定义了支撑：从这个列表中，去掉那些概率为 0 的值，剩下的就是支撑。

所以支撑是列表的"有效子集"——那些真正有概率质量落在上面的值。

## 一个例子帮助理解

设 \( X \) 表示掷一枚公平骰子的点数。

- 列表可以是 \(\{1, 2, 3, 4, 5, 6\}\)，且 \( P(X \in \{1,2,3,4,5,6\}) = 1 \)，所以 \( X \) 是离散型随机变量。
- 支撑为 \(\{1, 2, 3, 4, 5, 6\}\)，因为每个值的概率都是 \( 1/6 > 0 \)。

如果列表是 \(\{1, 2, 3, 4, 5, 6, 7, 8, \ldots\}\)，也满足 \( P(X \in \text{这个列表}) = 1 \)，但支撑仍然是 \(\{1, 2, 3, 4, 5, 6\}\)，因为 \( P(X = 7) = 0 \)，7 不属于支撑。

## 支撑（Support）的精确定义

**支撑** 是随机变量 \( X \) 的所有**可能取到的值**的集合，更准确地说，是所有使得概率 \( P(X = x) > 0 \) 的 \( x \) 所组成的集合。

用集合语言写出来：

\[
\text{Supp}(X) = \{ x \in \mathbb{R} \mid P(X = x) > 0 \}
\]

### 核心要点

| 关键点 | 说明 |
|---|---|
| 支撑是值的集合，不是样本点的集合 | 支撑里的元素是实数（随机变量取到的数值），不是样本空间 \( \Omega \) 里的元素。 |
| 用概率 \( > 0 \) 来定义 | 即使某个值在理论上可能出现，但如果概率为 0，它就不属于支撑。 |
| 支撑必须是有限集或可数无穷集 | 这是离散型随机变量的要求。如果支撑是不可数集（如整个实数轴），则 \( X \) 是连续型随机变量。 |

### 区分两个容易混淆的集合

| 概念 | 定义 | 元素类型 | 例子（掷一枚骰子） |
|---|---|---|---|
| **样本空间 \( \Omega \)** | 所有可能结果的集合 | 结果（事件的基本单元） | \( \{1, 2, 3, 4, 5, 6\} \)（朝上的点数） |
| **支撑 \( \text{Supp}(X) \)** | 随机变量 \( X \) 所有可能取值的集合 | 实数 | 如果 \( X = \) 点数，支撑也是 \( \{1,2,3,4,5,6\} \)，但它们是**数字**，不是"结果" |

在掷骰子的例子中，样本空间和支撑在数值上看起来一样，但它们的**身份不同**：
- 样本空间里是"朝上点数为 1 的这个结果"（一个事件的基本单元）。
- 支撑里是"数字 1"（一个实数）。

### 为什么"支撑"很重要？

习题 10 的一个问题是：是否存在支撑为 <span class="math"><math><mo stretchy="false">{</mo><mn>1</mn><mo>,</mo><mn>2</mn><mo>,</mo><mn>3</mn><mo>,</mo><mo lspace="0">.</mo><mo lspace="0" rspace="0.1667em">.</mo><mo lspace="0" rspace="0">.</mo><mo stretchy="false">}</mo></math></span> 的离散型随机变量，使它所对应的概率质量函数在 n 处的值为 1/n（提示：参考级数相关的知识）。

这个问题可以分解为两步：

1. **支撑指定了哪些 \( n \) 需要考虑**：只有 \( n = 1, 2, 3, \ldots \)。
2. **PMF 在这些 \( n \) 上的取值给定**：\( p(n) = 1/n \)。
3. **检查归一性**：\( \sum_{n=1}^{\infty} p(n) \) 是否等于 1？

如果支撑是有限的（比如 \( \{1, 2, \ldots, 10\} \)），那么即使 \( p(n) = 1/n \)，我们也可以通过归一化常数来修正（因为有限和总是有限的）。但题目指定支撑是可数无穷集，所以必须检查无穷级数是否收敛到 1。

### 总结

| 问题 | 答案 |
|---|---|
| 支撑是什么？ | 所有使得 \( P(X = x) > 0 \) 的实数 \( x \) 的集合 |
| 支撑和样本空间的区别？ | 支撑在实数轴上，样本空间在原始结果空间里 |
