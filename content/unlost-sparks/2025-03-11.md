代数除法是指在代数学中，对含有变量的多项式进行除法运算的过程。与数字除法类似，代数除法旨在找到一个多项式（被除式）可以被另一个多项式（除式）整除的结果，这个结果通常包括商和余数。如果存在一个多项式 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 被另一个多项式 <span class="math"><math><mi>D</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 除，我们可以表示为：

<div class="math">
<math display="block"><mi>P</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mi>D</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo lspace="0.2222em" rspace="0.2222em">·</mo><mi>Q</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>+</mo><mi>R</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math>
</div>

这里，

- <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 是被除式（即要被除的多项式）；
- <span class="math"><math><mi>D</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 是除式（即用来除的多项式）；
- <span class="math"><math><mi>Q</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 是商（即除法的结果）；
- <span class="math"><math><mi>R</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 是余数（满足它的次数小于除式的次数或为0）。

根据上述定义，当 <span class="math"><math><mi>R</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo><mo>=</mo><mn>0</mn></math></span> 时，我们说 <span class="math"><math><mi>D</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 整除 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span>，或者说 <span class="math"><math><mi>P</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 能被 <span class="math"><math><mi>D</mi><mo stretchy="false">(</mo><mi>x</mi><mo stretchy="false">)</mo></math></span> 整除。

代数除法可以通过多种方式进行，比如长除法、合成除法（仅适用于线性除式）、因式分解等方法来简化计算过程。这些方法可以帮助解决更复杂的代数问题，如寻找多项式的根、简化表达式等。

## 长除法

长除法是一种用于解决多项式除法问题的方法，它与我们在小学时学习的数字长除法非常相似。长除法适用于任何多项式的除法操作，特别是当除式为一次项或更高次项的多项式时尤为有用。通过这种方法，我们可以找到商和余数。

### 长除法步骤

1. **排列多项式**：确保被除式（分子）和除式（分母）都按照变量的降幂顺序写好。如果缺少某一项的系数，可以用0补上。
<br>
2. **除法开始**：用被除式的最高次项除以除式的最高次项，得到的结果是商的第一项。

3. **乘法和减法**：将上一步得到的商的第一项乘以整个除式，然后从被除式中减去这个乘积，注意对齐相同次数的项。

4. **重复过程**：将上述步骤应用于所得的新多项式（即上一步的差），直到新多项式的最高次项的次数小于除式的最高次项的次数为止。此时，你得到的就是余数。

5. **结果表达**：最终的结果包括商和余数两部分。如果余数为零，则表示被除式可以被除式整除；否则，结果通常表示为商加上余数除以除式的形式。

### 示例

假设我们要用长除法计算

<math display="block"><mo stretchy="false">(</mo><msup><mi>x</mi><mn>3</mn></msup><mo>−</mo><mn>12</mn><msup><mi>x</mi><mn>2</mn></msup><mo>−</mo><mn>42</mn><mo stretchy="false">)</mo><mi>÷</mi><mo stretchy="false">(</mo><mi>x</mi><mo>−</mo><mn>3</mn><mo stretchy="false">)</mo></math>
</div>

1. 将 <span class="math"><math><msup><mi>x</mi><mn>3</mn></msup><mo>−</mo><mn>12</mn><msup><mi>x</mi><mn>2</mn></msup><mo>+</mo><mn>0</mn><mi>x</mi><mo>−</mo><mn>42</mn></math></span> 按照 <span class="math"><math><mi>x</mi></math></span> 的降幂排列，并准备好除式 <span class="math"><math><mi>x</mi><mo>−</mo><mn>3</mn></math></span>。
2. 第一步：<span class="math"><math><msup><mi>x</mi><mn>3</mn></msup><mo lspace="0" rspace="0">/</mo><mi>x</mi><mo>=</mo><msup><mi>x</mi><mn>2</mn></msup></math></span>。这是商的第一项。
3. 接下来，将 <span class="math"><math><msup><mi>x</mi><mn>2</mn></msup></math></span> 乘以 <span class="math"><math><mi>x</mi><mo>−</mo><mn>3</mn></math></span> 得到 <span class="math"><math><msup><mi>x</mi><mn>3</mn></msup><mo>−</mo><mn>3</mn><msup><mi>x</mi><mn>2</mn></msup></math></span>，并将其从原多项式中减去，得到 <span class="math"><math><mo lspace="0" rspace="0">−</mo><mn>9</mn><msup><mi>x</mi><mn>2</mn></msup><mo>+</mo><mn>0</mn><mi>x</mi><mo>−</mo><mn>42</mn></math></span>。
4. 重复该过程，直至所有项都被处理过，最后得到商和余数。
